# DRL-SISR-HCVRP-Optimization
Solving Heterogeneous Capacitated Vehicle Routing Problem (HCVRP) using a hybrid approach of Deep Reinforcement Learning and SISR heuristic algorithm.
# Hybrid DRL-SISR for Heterogeneous VRP

## Overview
本專案為我的碩論實作，提出一套混合型演算法，旨在解決HCVRP。本方法將 CNN-based 深度強化學習 整合至 SISR (Slack Induction by String Removals) 啟發式演算法中，利用 AI 輔助決策，達成比傳統演算法更佳的搜尋效率與解品質。

## Tool
Language: Python

Frameworks: PyTorch, NumPy, Pandas, OpenAI Gym
